# LumoraKit Documentation

> **Work in Progress**

Welcome to the official documentation for **LumoraKit**.

LumoraKit is a premium, closed-source SwiftUI SDK that generates adaptive, artwork-driven backgrounds from a single image.

The SDK is designed to provide a clean public API while keeping the rendering engine private.

---

# Overview

LumoraKit analyzes an input image, extracts its dominant colors, builds an adaptive palette, and renders beautiful gradients that can be used as dynamic application backgrounds.

The long-term vision is to deliver Apple-quality adaptive rendering with a simple SwiftUI interface.

---

# Quick Start

```swift
import SwiftUI
import LumoraKit

struct ContentView: View {

    let artwork: UIImage

    var body: some View {
        LumoraView(
            image: artwork,
            gradient: .mesh
        )
    }
}
```

---

# Available Gradient Types

LumoraKit currently supports the following gradient renderers:

- Linear
- Radial
- Angular
- Mesh
- Organic

Each renderer uses the adaptive palette extracted from the supplied image.

# Architecture

LumoraKit follows a **two-repository architecture**.

The public repository is responsible for package distribution and documentation.

The rendering engine lives in a separate private repository.

```text
                    Public Repository

                    README.md
                    Docs.md
                    Package.swift
                    GitHub Releases
                           │
                           ▼
                Swift Package Manager
                           │
                           ▼
                 LumoraKit.xcframework
                           │
                           ▼
                Private Rendering Engine
```

This allows the public API to remain clean while protecting the implementation.

---

# Private Engine Structure

The rendering engine is organized into feature-focused modules.

```text
Sources
└── LumoraEngine
    ├── Public
    │   ├── LumoraView
    │   ├── LumoraGradient
    │   ├── LumoraStyle
    │   ├── LumoraConfiguration
    │   └── LumoraAnimation
    │
    └── Internal
        ├── Analysis
        │   ├── Image Analysis
        │   ├── Pixel Processing
        │   └── Color Space Conversion
        │
        ├── Palette
        │   ├── Histogram Extraction
        │   ├── K-Means Clustering
        │   ├── Palette Selection
        │   └── Color Utilities
        │
        ├── Gradient
        │   ├── Linear
        │   ├── Radial
        │   ├── Angular
        │   ├── Mesh
        │   └── Organic
        │
        ├── Pipeline
        ├── Animation
        ├── Rendering
        ├── State
        ├── Cache
        ├── Utilities
        └── Debug
```

The implementation is intentionally modular to keep each responsibility isolated and maintainable.

# Rendering Pipeline

Every background generated by LumoraKit follows the same high-level rendering pipeline.

```text
UIImage
    │
    ▼
Image Analysis
    │
    ▼
Color Extraction
    │
    ▼
Palette Generation
    │
    ▼
Gradient Renderer
    │
    ▼
SwiftUI View
```

This separation allows new rendering styles to be added without affecting the public API.

# Why Closed Source?

LumoraKit is distributed as a binary SDK.

The public repository exposes only the package interface, documentation, and release artifacts.

The rendering engine, adaptive color extraction algorithms, rendering pipeline, and implementation details remain proprietary.

This approach provides:

- Stable public API
- Faster integration
- Protected intellectual property
- Independent evolution of the rendering engine
